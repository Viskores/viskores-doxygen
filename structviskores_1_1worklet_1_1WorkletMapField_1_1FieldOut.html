Detailed Description

A control signature tag for output fields.

A FieldOut argument expects a viskores::cont::ArrayHandle in the associated parameter of the invoke. The array is resized before scheduling begins, and each invocation of the worklet sets a single value in the array.

This tag means that the field is write only.

Although uncommon, it is possible to set the worklet's InputDomain to a FieldOut argument. If this is the case, then the viskores::cont::ArrayHandle passed as the argument must be allocated before being passed to the invoke, and the input domain will be the size of the array.
