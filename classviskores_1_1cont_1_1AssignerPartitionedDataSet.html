Detailed Description

Assigner for PartitionedDataSet partitions.

AssignerPartitionedDataSet is a viskoresdiy::StaticAssigner implementation that uses PartitionedDataSet's partition distribution to build global-id/rank associations needed for several diy operations. It uses a contiguous assignment strategy to map partitions to global ids, i.e. partitions on rank 0 come first, then rank 1, etc. Any rank may have 0 partitions.

AssignerPartitionedDataSet uses collectives in the constructor hence it is essential it gets created on all ranks irrespective of whether the rank has any partitions.
