Detailed Description

ControlSignature tag for whole output arrays.

The WholeArrayOut control signature tag specifies an viskores::cont::ArrayHandle passed to the invoke of the worklet. An array portal capable of writing to any place in the array is given to the worklet. Developers should take care when using writable whole arrays as introducing race conditions is possible.
