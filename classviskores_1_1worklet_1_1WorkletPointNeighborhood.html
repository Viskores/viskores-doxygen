Detailed Description

Base class for worklets that map over the points in a structured grid with neighborhood information.

The domain of a WorkletPointNeighborhood is a viskores::cont::CellSetStructured. It visits all the points in the mesh and provides access to the point field values of the visited point and the field values of the nearby connected neighborhood of a prescribed size.
