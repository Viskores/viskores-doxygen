Detailed Description

Reading images using HDF5 Image format.

ImageReaderHDF5 extends viskores::io::ImageWriterBase and implements writing image HDF5 file format. It conforms to the HDF5 Image Specification https://portal.hdfgroup.org/display/HDF5/HDF5+Image+and+Palette+Specification%2C+Version+1.2
