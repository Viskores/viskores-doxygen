Detailed Description

A control signature tag for input values associated with the keys.

A WorkletReduceByKey operates by collecting all values associated with identical keys and then giving the worklet a Vec-like object containing all values with a matching key. This tag specifies an viskores::cont::ArrayHandle object that holds the values. The number of values in this array must be equal to the size of the array used with the KeysIn argument.
